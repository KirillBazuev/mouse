Инструкция по использованию программы:

1. Запустить файл moving.exe
	Откроется консоль, в которой будет текст: "Введите название видео (полностью, с расширением):"
2. Введите название видео в консоль
	Видео должно располагаться в папке videos. При вводе названия обязательно учитывать регистр, и указывать расширение (например: "mouse.MOV")
3. После обработки указанного видео, откроется кадр из видео, на котором нужно кликнуть левой кнопкой мыши по центральной норке для ввода ее координат.
4. После обработки, полученные данные сохраняются в папку data в файле Excel, а также открывается покадровый просмотр видео (номер кадра написан в заголовке окна в формате Frame = ...)
	Управление покадровым просмотром видео:
	"n" - Следующий кадр
	"p" - Предыдущий кадр
	"q" - Выйти
	Для управления важно, чтобы стояла латинская раскладка, и был выключен CapsLock.
	Также, в консоли будут выводится координаты курсора
